Helpful Hints:

ASR 920 guidelines:

  • Only add entries to interfaces we are using for your customer
  • If the generate button doesn't work, set all interfaces to PRI and then reselect the appropriate service for your interface
  • Don't attempt to drop the whole config at once, pace yourself

SAR-F to SAR-M guidelines:

  • Update the below to your customers relevant interfaces and the device name
  • You must use SAM5620 to get the 7705 SAR-F configuration if you haven't pulled it from the existing device

  • 1. Log into the Sam Client.
    2. Click on Administration>>NE Maintenance>>Backup/Restore.
    3. Click on the Backup/Restore Status Tab.
    4. In the Name column, enter the hostname and click on SEARCH.
    5. Double-Click on the entry that populates, a new dialogue box opens up.
    6. Click on the Back up tab.
    7. Select the most recent back up.
    8. Click on Export.
    9. Save to your PC (The Folder name is the IP of the Host).
    10. Drill down to the CF3 folder, you will find the routerâ€™s configuration file and its current BOF.cfg file as well.
